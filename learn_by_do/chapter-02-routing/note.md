# 第2章：路由（Routing）笔记

## 路由模式概述

- 路由将条件逻辑引入 Agent 的操作框架，使其能够从固定的执行路径转变为这样一种模型：Agent 动态评估特定标准，从一组可能的后续操作中进行选择。这使得系统行为更加灵活且具有上下文感知能力。
- 路由模式的几种实现

1. **基于 LLM 的路由** ： 利用语言模型分析输入并输出指示下一步操作的指令或标识符。
2. **基于嵌入的路由** ： 将输入转换为向量嵌入，并与不同路由的嵌入进行比较，选择最相似的路由。
3. **基于规则的路由** ： 使用预定义的规则或逻辑（如 if-else 语句）进行决策。
4. **基于机器学习模型的路由** ： 使用训练有监督模型的分类器，根据输入进行路由决策。
